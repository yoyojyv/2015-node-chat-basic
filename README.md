# 2015-node-chat-basic
=====================================
> A basic chatroom, based on the one in chapter 2 of Node.js in Action


### 01. 준비 
* node.js 설치
* package.json 파일 작성

```json
{
  "name": "nodechat", 
  "version": "0.0.1", 
  "description": "Simple multiroom chat server",
  "dependencies": {
    "socket.io": "~1.3.5", 
    "mime": "~1.3.4"
  }
}
```

* 콘솔창에서 다음의 명령어 입력 
```
$ npm install
```
